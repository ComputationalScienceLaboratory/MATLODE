Installation (Recommended)

  1. Download MATLODE.mltbx
  2. Navigate to MATLODE.mltbx in directory
  3. Right click on MATLODE.mltbx
  4. Click install

Alternative Installation

  1. Download MATLODE.tar.gz
  2. Navigate to MATLODE.tar.gz
  3. Extract all files
  4. Right click on root folder MATLODE
  5. Click Add to Path -> Selected Folders and Subfolders

Uninstall (Recommended)

  1. Click Add-Ons -> Manage Custom Toolboxes -> Uninstall MATLODE

NOTE: Only applies if MATLODE.tltbx is installed as a custom MATLAB toolbox.

Alternative Uninstall Method 1

  1. Naviage to MATLAB/Toolboxes
  2. Delete MATLODE
  3. Delete all MATLODE/* paths from MATLAB/Toolboxes/toolboxFolders.txt

NOTE: Only applies if MATLODE.tltbx is installed as a custom MATLAB toolbox.

Alternative Uninstall Method 2

  1. Naviage to MATLODE root folder
  2. Right click MATLODE root folder
  3. Click Remove from Path -> Selected Folders and Subfolders

NOTE: Only applies if MATLODE is installed by adding MATLODE to the current matlabpath.

Troubleshooting

Included in MATLODE are example scripts for every integrator. To troubleshoot a specific integrator, execute the appropriate script listed in the command window. To edit the script, execut 'edit scriptFileName' in the command window.

All Forward Integrators

  • Testing_Simple_FWD

All Tangent Linear Integrators

  • Testing_Simple_TLM

All Adjoint Integrators

  • Testing_Simple_ADJ

Simple Example Problem

Execute the following commands into the command window to run Van Der Pol using a forward explicit Runge-Kutta intregration scheme.

Ode_Function        = @arenstorfOrbit_Function;
Time_Interval       = [ 0 17.0652166 ];
Y0                  = [0.994; 0; 0; -2.00158510637908252240537862224];

[ ~, Y ] = MATLODE_ERK_FWD_Integrator(Ode_Function,Time_Interval,Y0)

Y =

    0.9940         0         0   -2.0016
    0.9894   -0.0081   -1.1139   -1.3474
